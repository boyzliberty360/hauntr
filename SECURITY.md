# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| 0.1.x | Yes |

## Reporting a Vulnerability

Please do **not** open a public GitHub issue for security vulnerabilities.

Instead, email: security@hauntr.dev (or open a [GitHub private advisory](https://github.com/boyzliberty360/hauntr/security/advisories/new))

You can expect a response within 48 hours. If the issue is confirmed, a patch will be released as soon as possible.
